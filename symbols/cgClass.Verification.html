cgClass.Verification (options)

描述：验证插件 demo

继承自： cgClass, cgClass.Ajax, jQuery

源码： cgClass.verification.js

作者：徐晨

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • btn {jQuery Object} 触发提交事件的按钮
  • error {Function} 验证错误时候的附加事件
  • errorTemp {Function} 返回失败提示的html片段
  • hookDom {css class} 被验证项的钩子名称
  • init {css class} 验证前的附加事件
  • inputWrap {css class} 被验证项的单行包裹，提示验证结果用
  • map {Object} 弹出框的种类
  • otherInput {Function} 特殊input的钩子名称
  • parseAjaxData {Object} 附加ajax data
  • success {Function} 验证成功时候的附加事件
  • successTemp {String} 返回成功提示的html片段
  • wrap {jQuery Object} 需要验证的表格包裹的jquery dom

Static Properties

Events

  • abortAjax () 停止当前验证控件中所有ajax事件
  • doMatchResult (event) 全局验证结束时的回调函数，成功触发附加成功事件，反之触发失败事件并传出错误的dom
  • match (val, name, ver, dom) 验证单条条件
  • matchAll (event) 触发当前验证控件中所有验证事件
  • numLast (val, msg, dom, range) 验证条件：被验证数据为数字且小数点后为{n-m}位 例:numLast,{1-2}
  • numRange (val, msg, dom, range) 验证条件：被验证数据为数字且在{n-m}范围中 例:numRange,{100-200}
  • parseVer (data, val, dom) 拆分验证条件
  • range (val, msg, dom, url, label, id) 验证条件：是否存在 例:isHave
  • thrown (dom, msg) 验证失败
  • verified (dom) 验证成功
  • waiting (dom) 验证等待中，验证方式为ajax时应用

Events Detail

  • abortAjax ()
    停止当前验证控件中所有ajax事件
  • doMatchResult (event)
    全局验证结束时的回调函数，成功触发附加成功事件，反之触发失败事件并传出错误的dom
    参数：
    •   event {Object} 如果为验证按钮触发，传入当前按钮的event对象
  • match (val, name, ver, dom)
    验证单条条件
    参数：
    •   val {String} 被验证数据
    •   name {String} 验证信息名称
    •   ver {String} 单条验证信息
    •   dom {jQuery Object} 被验证jQuery Object
  • matchAll (event)
    触发当前验证控件中所有验证事件
    参数：
    •   event {Object} 如果为验证按钮触发，传入当前按钮的event对象
  • {Object} numLast (val, msg, dom, range)
    验证条件：被验证数据为数字且小数点后为{n-m}位 例:numLast,{1-2}
    参数：
    •   val {String} 被验证数据
    •   msg {String} 错误信息
    •   dom {jQuery Object} 被验证jQuery Object
    •   range {String} 范围值
    Returns
    • {Object} { result : true|false, msg : msg };
  • {Object} numRange (val, msg, dom, range)
    验证条件：被验证数据为数字且在{n-m}范围中 例:numRange,{100-200}
    参数：
    •   val {String} 被验证数据
    •   msg {String} 错误信息
    •   dom {jQuery Object} 被验证jQuery Object
    •   range {String} 范围值
    Returns
    • {Object} { result : true|false, msg : msg };
  • parseVer (data, val, dom)
    拆分验证条件
    参数：
    •   data {String} 全部验证信息 格式为xxx:xxx/xxx 注:/为条件分隔符
    •   val {String} 被验证数据
    •   dom {jQuery Object} 被验证jQuery Object
  • {Object} range (val, msg, dom, url, label, id)
    验证条件：是否存在 例:isHave
    参数：
    •   val {String} 被验证数据
    •   msg {String} 错误信息
    •   dom {jQuery Object} 被验证jQuery Object
    •   url {String} 隐藏参数 ajax url 为dom的data-same-url属性的值
    •   label {String} 隐藏参数 ajax data.label 值为被验证数据,不可修改
    •   id {String} 隐藏参数 ajax data.id 为dom的data-same-id
    Returns
    • {Object} { result : "ajax", msg : msg };
  • thrown (dom, msg)
    验证失败
    参数：
    •   dom {jQuery Object} 被验证jQuery Object
    •   msg {String} 输出的错误提示
  • verified (dom)
    验证成功
    参数：
    •   dom {jQuery Object} 被验证jQuery Object
  • waiting (dom)
    验证等待中，验证方式为ajax时应用
    参数：
    •   dom {jQuery Object} 被验证jQuery Object
