cgClass.Autocomplete (options)

描述：自动完成插件 demo

继承自： cgClass, cgClass.Ajax, jQuery, jQuery.getFromTemplate

源码： cgClass.Autocomplete.js

作者：徐晨

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • autoBtn {jQuery dom} 提交按钮的jquery dom
  • autoItem {jQuery class} 展示项的钩子名称
  • autoList {jQuery class} 展示列表的钩子名称
  • autoText {jQuery class} 输入框的钩子名称
  • hasBtn {Boolean} 是否存在提交按钮
  • hoverClass {css class} 悬停显示项时添加的样式名
  • template {String html} 展示列表的钩子名称
  • wrap {jQuery Object} 需要验证的表格包裹的jquery dom

Static Properties

Methods

  • action (option, val) 表单提交的事件
  • createSearchList (option, val, addItem) 生成显示列表的事件
  • init (option) 模块初始化时的附加事件

Events

  • addItem (data) 添加展示项
  • closeList () 隐藏显示列表
  • returnText (value) 替换输出框的值为带入参数
  • showList () 显示显示列表，并绑定隐藏列表功能
